

Core Features

Add Books – Users can add books with details such as title, author, ISBN, price, genre, and description.
View Books – Books are displayed dynamically in a card-based format.
Edit Books – Users can update book details using a pop-up modal.
Delete Books – Books can be removed with a confirmation prompt.
Read More – View additional details of a book in a pop-up modal.
Chatbot Integration – Interactive chatbot feature for user assistance.

Technologies Used

Frontend: HTML, CSS, JavaScript
Backend: Firebase Firestore (NoSQL Database)
UI Libraries: Unicons, FontAwesome
Chatbot: GeminAi AI
